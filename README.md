# Testing SSE

Just a simple app for trying SSE

## Server.go

Go to the server

```bash
cd server
```

run the server

```bash
go run main.go
```

Server will be linstening on port:8080

## Client App

Go to the client app

```bash
cd www
```

install dependecies

```bash
pnpm i
```

then run the app

```bash
pnpm dev
```
